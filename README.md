# Frontend Mentor - Contact Form (Development Guide)

This repository is used to build the Contact Form challenge from Frontend Mentor using a clear and professional workflow.

## 🎯 Goal

- Build a responsive contact form
- Practice HTML and CSS
- Learn how to work like a real developer using Git and GitHub


## 🧠 How I Work

I follow a simple process:

1. Create an issue (task)
2. Create a feature branch from `develop`
3. Work only on that task
4. Commit changes
5. Push the branch
6. Create a Pull Request → `develop`
7. Merge after review
8. When finished → merge `develop` → `main`


## 🌿 Branches

- `main` → final version (clean and approved)
- `develop` → working branch
- `feature/*` → one branch per task

### Example:
- `feature/form-structure`
- `feature/form-styling`
- `feature/form-validation`
- `feature/responsive`


## 📋 Tasks (Issues)

Each task is created as an issue on GitHub.

Rules:
- One issue = one task
- One branch = one issue
- One Pull Request = one task
